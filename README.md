# Decent Glam Shopping Website

A full-stack e-commerce website developed for online shopping and product management.

## Features
- Product listing
- Product details
- User-friendly interface
- Database integration
- CRUD operations
- Responsive design

## Technologies Used
- JavaScript
- Node.js
- Express.js
- MongoDB
- HTML
- CSS

## Purpose
This project was developed to provide an online shopping platform with efficient product management and a smooth user experience.
